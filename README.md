# 🩺 당뇨병 가이드 반응형 웹사이트 (Diabetes Guide)

당뇨병의 위험성을 인지하고, 예방 수칙 및 건강한 식단을 가이드해 주는 반응형 웹사이트입니다.

## ✨ 주요 기능
- **메인 홈**: 당뇨 관리의 중요성과 핵심 가이드를 한눈에 확인
- **위험성 및 예방**: 당뇨 합병증의 위험성과 생활 속 예방 수칙(체중 관리, 운동 등) 정보 제공
- **맞춤형 식단 가이드**: 아침, 점심, 저녁, 간식 등 시간대별 권장 식단 정보 및 필터링 기능 제공
- **반응형 디자인**: 모바일, 태블릿, 데스크톱 등 모든 기기에서 최적화된 화면 제공

## 🛠 기술 스택
- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Routing**: React Router DOM
- **Icons**: Lucide React

## 🚀 시작하기

### 1. 저장소 복제
```bash
git clone <repository-url>
cd diabetes-guide
```

### 2. 의존성 설치
```bash
npm install
```

### 3. 로컬 서버 실행
```bash
npm run dev
```

### 4. 빌드
```bash
npm run build
```

## 📄 참고 사항
본 웹사이트에서 제공하는 정보는 일반적인 건강 정보를 목적으로 하며, 전문적인 의학적 진단이나 치료를 대신할 수 없습니다. 정확한 진단은 반드시 전문의와 상담하시기 바랍니다.
